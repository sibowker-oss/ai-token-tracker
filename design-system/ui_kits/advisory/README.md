# Advisory UI Kit

Light, executive kit for **Hepburn Advisory** (`hepburnadvisory.com.au`).

## What's in here
- `index.html` — single-page site kit: nav, hero, proof strip, two-track services, About strip, Ledger teaser (dark band), Insights grid, footer.

## Key patterns
- **Hero** — first-person singular copy ("I advise…"). No imagery. Display type dominates.
- **Two-track services** — exactly two cards, equal weight: Founder Advisory (blue pill) and Executive Advisory (amber pill). Never a third track. Never "Fractional CXO".
- **Proof strip** — three numbers only: years, cumulative ARR, engagement length. No logos (no fake client implication).
- **About** — pull-quote in Source Serif italic; credits in a 2×2 grid; portrait placeholder (drop a real photo into `assets/portrait.jpg`).
- **Ledger strip** — inverse dark band teasing the publication. The only place on the Advisory site where the gradient clip-text hook appears.
- **Insights** — kicker `[format] · [reading time]` (Essay / Brief / Letter), serif headline feel, read-more link with top border.
- **Footer** — three-column, with email as primary action.

## Non-negotiables
- No "we". No "Fractional". No client logos. No fake testimonials.
- Max width 1200px. Warm off-white page bg. Crisp white cards. No shadows beyond `--shadow-sm`.
