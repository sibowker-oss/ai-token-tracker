Methodology & Sources

Provider revenue: OpenAI ($16.8B ARR est. Q1'26 — WSJ, Bloomberg); Anthropic ($14B ARR Mar'26 — SaaStr); Google Cloud AI ("billions" — Alphabet earnings); xAI, DeepSeek, Mistral from press reports.
Token volumes: Triangulated from 4 independent methods: (1) Revenue ÷ blended pricing, (2) GPU capacity ceiling (~4.5M H100-equiv × 30-50% utilization — SemiAnalysis), (3) Bottom-up user × session × token estimation, (4) SDK download proxies (openai PyPI: 224M/month, anthropic: 77M/month — pypistats.org, Mar 2026). Previous estimate of ~680T/day revised down to ~565T based on GPU utilization cross-check.
Self-hosted tokens: Not included in provider totals. Llama 3.1 8B: 8.3M HuggingFace downloads/30d. vllm: 7.4M PyPI downloads/month. Estimated ~80-120T/day of inference happening outside API providers — real usage generating zero revenue.
Skeptical cross-checks: Sequoia "$200B question" capex gap (David Cahn, Sep 2024). Goldman Sachs "Too much spend, too little benefit" (Jun 2024). UBS: 4.4% of US adults use AI daily. ChatGPT Day-30 retention: 10-15% (Sensor Tower). GPU utilization 30-50% (SemiAnalysis). 85% AI projects fail (RAND/Gartner).
Enterprise reality: Growth deceleration data from earnings calls (SF: 18%→8-9%, NOW: 25%→20%, MSFT M365: stable, WDAY: 20%→15-17%). SKU bundling analysis from UpperEdge. M365 Copilot usage 30-40% (Gartner, Directions on Microsoft).
Geographic/Industry: Stanford HAI AI Index 2024, IDC, SlashData, Gartner AI Use Case Survey 2024, McKinsey Global AI Survey 2024.
Live data: OpenRouter pricing via public API (openrouter.ai/api/v1/models). PyPI/npm download stats from pypistats.org and api.npmjs.org. HuggingFace model downloads via API.
Cross-dimensional weights: Editorial estimates for filtering. Filtered views show directional splits, not precise figures.
